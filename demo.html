jQuery Easing Repeater plugin

Demonstration

This is a customisable demonstration of how the an easing repeater works. The controls on the left let you pick which easing function to use and the repitition parameters, while the options on the right select some basic CSS properties to animate on the blue box below.

The graph gives a visualisation of the original easing equation in white and the new repeated equation in yellow. (Easing graph code shamelessly stolen from James Padolsey's Easing Illustrated page)
